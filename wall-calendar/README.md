# 📅 Wall Calendar App (Vite + React)

A simple and responsive wall calendar application built using **React** and **Vite**.

---

## 🚀 Features
- 📆 Interactive calendar UI
- ⚡ Fast performance with Vite
- 🎨 Clean and responsive design
- 🔧 Easy to customize

---

## 🛠️ Tech Stack
- React
- Vite
- JavaScript (ES6+)
- CSS

---
